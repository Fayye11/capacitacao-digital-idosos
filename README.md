# Capacitação Digital: Workshop de Autonomia Tecnológica para a Terceira Idade

Este repositório centraliza a documentação pedagógica e o planejamento da Atividade Extensionista I, desenvolvida para o curso de Análise e Desenvolvimento de Sistemas do Centro Universitário Internacional UNINTER.

## 📌 Escopo do Projeto
O projeto consistiu em um workshop presencial voltado para a inclusão digital de idosos na comunidade de Luziânia/GO, focado em autonomia, acessibilidade e segurança da informação.

## 📂 Conteúdo Programático Aplicado
* **Módulo WhatsApp:** Configuração de privacidade, chamadas de vídeo, envio de mídias e gerenciamento de armazenamento.
* **Módulo Segurança Digital:** Identificação de engenharia social, verificação em duas etapas, comportamento preventivo contra golpes e links maliciosos.
* **Módulo E-Gov:** Navegação guiada e autenticação segura na plataforma Gov.br para acesso a direitos e benefícios.

## 🛠️ Tecnologias de Apoio e Referência
* Sistema Operacional Android / iOS (Dispositivos móveis dos usuários)
* Plataforma de Autenticação Federal (App Gov.br)
* Aplicativo de Mensagens Instantâneas (WhatsApp)
* Recursos de Espelhamento de Tela para Fins Didáticos
